What you get

Complete R7RS

Every identifier from Appendix A. 554 built-in procedures, 32 syntax forms, all 14 standard libraries.

First-class continuations

Multi-shot call/cc via stack copying. dynamic-wind, call/ec for escape continuations.

JIT compiler

Hot functions compile to native AArch64. Inline arithmetic, comparisons, car/cdr, cons.

51 SRFIs

8 built-in, 43 portable. Lists, strings, hash tables, vectors, threads, formatting, char-sets, and more.

C FFI

Call shared libraries from Scheme. Supports callbacks — pass Scheme procedures to C functions expecting function pointers.

Green threads

Fiber-based concurrency with channels. Full SRFI-18 compatibility: threads, mutexes, condition variables.
